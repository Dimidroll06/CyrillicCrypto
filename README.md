# CyrillicCrypto

Crypt and decrypt cyrillic text

Выберите тип функции:
-----------------------------------

c/с - crypt (Зашифровать текст);
d/в - decrypt(Дешифровать текст);

-----------------------------------

exemple using crypt: 2: Привет мир!
output: 2: сткдзф окт!

-----------------------------------
exemple using decrypt: 2: сткдзф окт!
output: 2: привет мир!
